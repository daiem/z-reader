---

## 🎉 功能

- [x] 支持格式:txt、epub
- [x] 支持起点网小说搜索和阅读
- [x] 支持笔趣阁小说搜索和阅读
- [x] 支持侧边栏章节显示(epub、在线)
- [x] 支持热键、菜单
- [x] 自定义阅读界面的 CSS 和 HTML
- [x] 状态栏进度显示
- [x] 阅读进度自动保存
- [x] 阅读进度跳转
